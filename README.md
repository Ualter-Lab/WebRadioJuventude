📻 Web Rádio Juventude

Site institucional desenvolvido para a Web Rádio Juventude, com foco em apresentar a rádio, facilitar o contato com ouvintes e anunciantes e oferecer uma experiência moderna, rápida e totalmente responsiva.

✨ Funcionalidades

* 🎵 Design moderno e intuitivo
* 📱 Layout 100% responsivo
* ⚡ Navegação com rolagem suave
* 🎨 Interface desenvolvida com HTML, CSS e JavaScript
* 📞 Área de contato
* 📢 Seção dedicada para anunciantes
* 🖼️ Ícones e favicon personalizados
* 🚀 Alto desempenho e carregamento rápido

🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

📂 Estrutura do projeto

WebRadioJuventude/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   ├── icons/
│   └── favicon/
└── README.md

A estrutura pode variar conforme a organização do projeto.

🚀 Como executar

1. Clone o repositório:

git clone https://github.com/Ualter-Lab/WebRadioJuventude.git

2. Acesse a pasta do projeto:

cd WebRadioJuventude

3. Abra o arquivo index.html em qualquer navegador.

Ou utilize uma extensão como Live Server no VS Code para uma melhor experiência durante o desenvolvimento.

🌐 Objetivo

O projeto foi desenvolvido para fortalecer a presença digital da Web Rádio Juventude, oferecendo um ambiente moderno para apresentação da rádio, divulgação de informações e aproximação com ouvintes e parceiros comerciais.
